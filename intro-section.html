I'm a web developer with a knack for design. I code in Philly—where I've lived my whole life. I also build custom effects pedals under the banner of Sixty/Forty Custom Pedals and play guitar/sing in my noiserock/post-hardcore band FAKING

I built this site utilizing Jekyll as a simple framework - writing in HAML, SASS, and jQuery - using partial file structure for both markup and stylesheets - while untilizing BEM namespacing and elements of ITCSS. I used Jekyll as a simple framework so that I can host the site on GitHub Pages.
